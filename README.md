## 🚀 The Experience
<div align="left">

"อ่านปะ" ถูกพัฒนาขึ้นเพื่อแก้ปัญหาความยุ่งยากในการจองหนังสือภายในองค์กร โดยเน้น **"Zero Friction Design"** ที่ทุกคนใน Botnoi สามารถใช้งานได้ทันทีโดยไม่ต้องเรียนรู้ใหม่

### ⚡️ Why Arn Pa?
* **Real-time Availability:** เห็นสถานะการจองทันที ไม่ต้องถามแอดมิน
* **Visual-First UI:** อินเทอร์เฟซที่ใช้การ์ดหนังสือสวยงาม พร้อมระบบจัดการรูปภาพอัจฉริยะ (Smart Fallback UI)
* **Glassmorphism Design:** ดีไซน์ล้ำสมัยด้วยเอฟเฟกต์ Glassmorphism สบายตา ใช้งานได้ยาวนาน
* **Robust Backend:** ข้อมูลแม่นยำ ปลอดภัย จัดการผ่าน Supabase อย่างเป็นระบบ
